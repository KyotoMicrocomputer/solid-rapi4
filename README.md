# SOLID for Raspberry Pi 4

## 必要なもの

- [Raspberry Pi 4 Model B](https://www.raspberrypi.com/products/raspberry-pi-4-model-b/)
- 容量16GB以上のSDカード
- ネットワーク接続
  - `Raspberry Pi 4 Model B` とホストPCがネットワークで通信できる必要があります。
